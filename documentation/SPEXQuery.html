Class: SPEXQuery

SPEXQuery

new SPEXQuery()

A SPEX query object, which represents a SPARQL query (inherited from the jQuery SPARQL class, see https://github.com/jgeldart/jquery-sparql), saves user selected (bound) variables and has additionally space and time expander methods (which add corresponding optional statements).
Properties:
Name Type Description
SPEXvariables object holds the variables that the user has generated in the queryPane
spatialConstraints object holds the spatial constraints on a node in focus.
temporalConstraints object holds the temporal constraints on a node in focus.
patterns object holds SPARQL patterns (triples, subgraphs, optionals).
filters object holds SPARQL filters .
prefixes object holds the list of SPEX prefixes.
timeout number number of seconds before timeout is thrown.
limit number maximum number of result items displayed
