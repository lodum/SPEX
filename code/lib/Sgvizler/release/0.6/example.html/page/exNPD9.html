sgvizler.visualization.Map: Things north of 72.5 degrees north

The element which draws the above chart:

<div id="sgvzl_example_query" 
   data-sgvizler-endpoint="http://sws.ifi.uio.no/sparql/npd" 
   data-sgvizler-query="SELECT ?lat ?long ?name &quot;&quot; ?url WHERE { ?url npdv:nsDecDeg ?lat ; npdv:ewDecDeg ?long ; npdv:name ?name ; FILTER (?lat > 72.5) }" 
   data-sgvizler-chart="sgvizler.visualization.Map" 
   data-sgvizler-loglevel="2" 
   style="width:800px; height:400px;"/>
